 - Be a space dwarf! Run [WASD] around a planet! 
 - See the sights [Mouse]! 
 - Smash [Left click] trees, sheeps, and mountains! 
 - Build [1-8][E] a home! Lift [F] heavy things! 
 - Fly [Space][Space]! 
 - Craft on a bench (literally)!
